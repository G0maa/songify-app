# Songify-app | api

- A Music recommendation website, this repo covers the API part.

## Tech stack

- NestJS
- Prisma
- PostgreSQL
- PassportJS
- ...

## Installation

```bash
$ yarn install
```

## Running the app

```bash
# development
$ yarn run start

# watch mode
$ yarn run start:dev

# production mode
$ yarn run start:prod
```

## Test

```bash
# API Testing
$ yarn run test
```
